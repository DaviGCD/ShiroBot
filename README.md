Shiro Bot
=========

Shiro é um bot focado em música para entreter você, ela é simples de se usar, entre as coisas dela tem:
- Sistema de música usando lavalink.
- Qualidade de áudio.
- Comandos super "novos".
- Personalidade fiel ao personagem original.


Criada usando
-------------
- Node.js
- discord.js
- lavalink
